# utils

Este diretório centraliza as funções utilitárias para o fluxo de execução da automação

`menu.py`:
Instancia os objetos do tipo Workflow, e exibe o menu para que o usuário escolha a opção.

`webdriver_setup.py`:
Define caracterísitcas do navegador que será usado para a automação, qual navegador será utilizado, em tela cheia, sem extensões e ignorando alertas de certificado.
